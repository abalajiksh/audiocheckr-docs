# Technical Reference

This section provides a comprehensive explanation of the digital signal processing (DSP) algorithms and mathematics that power AudioCheckr's detection capabilities.

## Overview

AudioCheckr uses pure DSP techniques—no machine learning—to detect audio quality issues. The core principle is that lossy compression and improper format conversion leave detectable artifacts in the audio signal.

## Key Algorithms Summary

### 1. Short-Time Fourier Transform (STFT)

The foundation of spectral analysis. We compute FFT over overlapping windows to analyze frequency content over time:

$$
X[k, m] = \sum_{n=0}^{N-1} x[n + mH] \cdot w[n] \cdot e^{-j2\pi kn/N}
$$

Where:
- $x[n]$ = input samples
- $w[n]$ = window function (Hann)
- $N$ = FFT size (4096)
- $H$ = hop size (1024)
- $m$ = frame index

### 2. Lossy Codec Detection

Lossy codecs remove high-frequency content. We detect this by:

1. Computing average magnitude spectrum across frames
2. Finding where spectrum drops significantly (30dB below peak)
3. Measuring rolloff steepness (dB/octave)
4. Matching against known codec signatures

### 3. Bit Depth Analysis

Fake 24-bit audio (upscaled from 16-bit) has telltale signs:

1. **LSB Analysis**: Trailing zeros in sample values
2. **Histogram Analysis**: Value clustering at 256-boundaries
3. **Quantization Noise**: Noise floor characteristics
4. **Entropy Analysis**: Lower 8-bit distribution

### 4. Upsampling Detection

Upsampled audio lacks genuine high-frequency content:

1. **Spectral Null Detection**: Energy void at original Nyquist
2. **Inter-sample Analysis**: True high-res has inter-sample peaks
3. **Nyquist Mirror Detection**: Aliasing artifacts

## Mathematical Notation

Throughout this documentation:

| Symbol | Meaning |
|--------|---------|
| $x[n]$ | Discrete-time signal sample at index $n$ |
| $X[k]$ | DFT coefficient at frequency bin $k$ |
| $f_s$ | Sample rate (Hz) |
| $f_N$ | Nyquist frequency = $f_s / 2$ |
| $N$ | FFT size / window length |
| $\|X[k]\|$ | Magnitude of complex FFT bin |
| $dB$ | Decibels: $20 \log_{10}(\text{ratio})$ |
