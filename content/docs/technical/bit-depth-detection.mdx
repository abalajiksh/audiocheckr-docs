# Bit Depth Detection

## Methods

1. **LSB Precision Analysis** - Trailing zeros
2. **Histogram Analysis** - Unique values
3. **Quantization Noise** - Noise floor
4. **Value Clustering** - Entropy

## Thresholds

| Bit Depth | Step Size |
|-----------|-----------|
| 16-bit | 1/32768 |
| 24-bit | 1/8388608 |
