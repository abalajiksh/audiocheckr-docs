# DSP Fundamentals

## Discrete Fourier Transform

The DFT converts time-domain signals to frequency domain:

```
X[k] = Σ x[n] · e^(-j2πkn/N)  for n = 0 to N-1
```

Where:
- X[k] is the frequency-domain representation at bin k
- x[n] is the time-domain sample at index n
- N is the FFT size

## FFT Size Selection

| Size | Resolution (44.1kHz) | Time Resolution |
|------|---------------------|-----------------|
| 1024 | 43.1 Hz | 23.2 ms |
| 2048 | 21.5 Hz | 46.4 ms |
| **4096** | **10.8 Hz** | **92.9 ms** |
| 8192 | 5.4 Hz | 185.8 ms |

## Frequency Resolution

Each FFT bin corresponds to a specific frequency:

```
f_k = (k × f_s) / N
```

Where f_s is the sample rate and N is the FFT size.

**Example** for 44.1kHz with FFT size 4096:
- Bin 0 = 0 Hz (DC)
- Bin 1 = 10.77 Hz
- Bin 2048 = 22050 Hz (Nyquist)

## Decibel Calculations

Amplitude to dB conversion:

```
L_dB = 20 × log₁₀(A / A_ref)
```

| dB | Linear Ratio | Meaning |
|----|--------------|---------|
| 0 dB | 1.0 | Full scale |
| -6 dB | 0.5 | Half amplitude |
| -20 dB | 0.1 | 10% amplitude |
| -96 dB | ~0.000016 | 16-bit noise floor |
| -144 dB | ~0.000000006 | 24-bit noise floor |

## Window Functions

### Hann Window (Default)

```
w[n] = 0.5 × (1 - cos(2πn/N))
```

Properties:
- Good frequency resolution
- -31.5 dB first sidelobe
- Commonly used for general spectral analysis
