# AudioCheckr

**AudioCheckr** is a sophisticated audio analysis tool that detects fake lossless files, transcoding artifacts, and quality issues using advanced DSP techniques.

## What Can AudioCheckr Detect?

- **Lossy Transcodes**: Identify MP3, AAC, Opus files converted to lossless
- **Fake Bit Depth**: Detect 16-bit audio masquerading as 24-bit
- **Upsampling**: Find audio upsampled from lower sample rates
- **Codec Artifacts**: Pre-echo, phase issues, joint stereo artifacts

## Quick Start

```bash
git clone https://github.com/abalajiksh/audiocheckr
cd audiocheckr
cargo build --release
./target/release/audiocheckr -i music.flac
```
